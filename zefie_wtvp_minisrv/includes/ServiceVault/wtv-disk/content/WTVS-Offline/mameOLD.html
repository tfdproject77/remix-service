M.A.M.E.
Once you start MAME you will stay in MAME, meaning everytime you turn on your WebTV MAME will start up. If you want to get back to HackTV Home you have 3 options.
  • Click the Exit Mame link on Mame Home.
  • Unplug for 10 seconds, plug it back in.
  • Use poweroff code 8673094 after you poweroff.
M.A.M.E. (an acronym of Multiple Arcade Machine Emulator) is an emulator application designed to recreate the hardware of arcade game systems in software on modern personal computers and other platforms. It was ported to WebTV unknown to subscribers in 1997 and provided arcade classics from the '80s. It was only ever intended for demonstration purposes.
